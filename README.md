# vlc-most-played

A script that outputs current playing tunes and keeps plays counter in a sqlite3 database.

## How to use

Clone the repo

```bash
git clone git@github.com:m4444te/vlc-most-played.git
```

Install dependencies

```bash
cd vlc-most-played
npm install
```

Run the script

```bash
npm start
```
