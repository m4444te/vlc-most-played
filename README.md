# vlc-most-played

A script that outputs current playing tunes and keeps plays counter in a sqlite3 database.

## How to use

Clone the repo, change directory and install dependencies

```bash
git clone https://github.com/m4444te/vlc-most-played.git
cd vlc-most-played
npm install

```

Run the script

```bash
npm start
```
